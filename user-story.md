# User Stories

## Create Account
As a customer  
I need to create an account  
So that I can use the service  

## Read Account
As a user  
I need to view an account  
So that I can see my details  

## Update Account
As a user  
I need to update my account  
So that my information is correct  

## Delete Account
As a user  
I need to delete an account  
So that I can remove unused data
