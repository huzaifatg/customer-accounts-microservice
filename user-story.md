# User Stories

**As a** customer  
**I need** to create an account  
**So that** I can use the service  

---

**As a** user  
**I need** to view accounts  
**So that** I can manage them  

---

**As a** user  
**I need** to update an account  
**So that** my information is current  

---

**As a** user  
**I need** to delete an account  
**So that** I can remove unused data
